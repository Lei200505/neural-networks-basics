# Logistic Regression from Scratch

A program egy logisztikus regressziós modellt valósít meg saját implementációval. Kétdimenziós mesterséges/véletlen adathalmazon tanít egy sigmoid aktivációs függvény, log loss veszteségfüggvény és gradiens alapú frissítés segítségével. Alapvető neurális haló modell.
